# client-api-lib
A Rust library to facilitate sending and receiving requests between the web client frontend and online backend.

### Setup
- Navigate into the tcp-client folder
  - `cd ./tcp-client`

- Download all library targets
  - `cargo build`

- Run the application
  - `cargo run`
